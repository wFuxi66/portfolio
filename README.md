# Portfolio - Jimmy ZHENG

🌐 **[Voir le portfolio](https://wfuxi66.github.io/portfolio/)**

Portfolio personnel développé avec React et Tailwind CSS, déployé automatiquement sur GitHub Pages.

## ✨ Features

- **Typing Effect** - Titre animé qui cycle entre différents rôles
- **Custom Cursor** - Curseur personnalisé avec anneau cyan
- **Smooth Scroll** - Navigation fluide entre les sections
- **Responsive Design** - Adapté mobile, tablette et desktop
- **Dark Theme** - Thème sombre avec accents cyan
- **OG Image** - Preview optimisé pour le partage sur les réseaux sociaux

## 🎮 Easter Egg

Tape le code Konami sur ton clavier : `↑ ↑ ↓ ↓ ← → ← → B A`

## 🛠️ Stack Technique

- **React** - Framework JavaScript
- **Tailwind CSS** - Styling
- **Vite** - Build tool
- **GitHub Actions** - CI/CD

## 🚀 Développement local

```bash
# Installation
npm install

# Lancer le serveur de dev
npm run dev

# Build production
npm run build
```
